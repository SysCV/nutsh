A mask represents pixel-level classification, where each pixel can be assigned to a specific class.
When two pixels share the same color in the canvas, they belong to the same class.
This allows for the identification of objects with complex shapes or compositions.
Follow the steps below to start drawing masks.

- Press <kbd>M</kbd> or select the "_Mask_" tool from the sidebar.
- Start drawing on the canvas. Take a look at the toolbar in the top-left corner.
- Press <kbd>Enter</kbd> to confirm.
