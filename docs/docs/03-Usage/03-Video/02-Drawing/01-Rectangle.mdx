Rectangles are useful for annotating object bounding boxes. To draw a rectangle, simply

- Press <kbd>A</kbd> or select the "_Rectangle_" tool from the sidebar.
- Click on the canvas to set one corner of the rectangle.
- Click somewhere else to define the opposite corner and complete the drawing.
